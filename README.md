# Rick and Morty app
This is a basic app build with `create-react-app` using typescript
Makes a request to [The Rick and Morty API](https://rickandmortyapi.com/) and shows a list.

Install the required modules<br>

`npm install` 

## Run the project <br>

###  `npm start`
